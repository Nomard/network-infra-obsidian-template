# Как воспользоваться шаблоном:
Точнее, как сделать так, чтобы заработало:
- скачиваем репозиторий
- переименовываем в корне каталог `obsidian` в `.obsidian` (на гит по дефолту низзя загружать .name файлы, они автоматом летят в gitignore)
- ставим обсидиан с офф-сайта
- при запуске обсидиан выбираем `Open folder as vault` и выбираем наш скачанный каталог `VaultSmaple`
- профит, должно всё заработать.
